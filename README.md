# coe-messages

共体験アプリケーションフレームワークで用いられるメッセージの型定義を提供します。

## ビルド

**coe-messages** はTypeScriptで書かれた型定義モジュールです。ビルドにはNode.jsが必要です。

`npm run build` によりビルドできます。

```sh
npm install
npm run build
```

## テスト

```sh
npm test
```
